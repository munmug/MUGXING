## Full test plan (all new functions)
Open in a browser:
- `…/functions/v1/market-pulse` → JSON ticker, gold `live:true`
- `…/functions/v1/stock-quote?symbol=TSLA` → real price + volatility, `live:true`
- `…/functions/v1/llm` → `{"error":"method_not_allowed"}` (POST-only; correct)
- `…/functions/v1/edgar?symbol=TSLA` → Tesla CIK, recent 10-K/10-Q filings,
  real revenue + net income by year, `live:true`

In the running app:
- Type `帮我分析微软的云业务` → routes to Guided Research, identifies Microsoft
  (Phase 3 — a subject the keyword router never knew).
- Run a Guided Research on a US name (e.g. TSLA/NVDA/AAPL) → the report is now
  generated by the LLM from real SEC filings, and the trace graph shows real
  EDGAR filings (Phase 4).
- Save a report, refresh → it persists (Phase 2).

## Honest scope notes
- EDGAR covers **US companies only**. Tencent (00700.HK) and other HK/CN names
  fall back to the original hardcoded report until HK retrieval is added.
- Phase 4 v1 synthesizes from structured XBRL financials + filing metadata, not
  full filing text. Deeper full-text reading is a later enhancement.
- Free LLM/data tiers are rate-limited — fine for testing, not production scale.
